당신은 교육용 수학 웹앱을 설계하고 구현하는 시니어 프론트엔드 개발자입니다.

React + TypeScript + Vite + Tailwind CSS를 사용해서 “변화율에서 편미분방정식까지”라는 수학 학습 웹앱을 만들어주세요.

이 앱은 단순 문제은행이 아니라, 개념 설명과 문제 풀이를 함께 제공하는 학습 앱입니다.

# 데이터 구조

모든 단원 데이터는 src/data/lessons.ts에 TypeScript 배열로 작성해주세요.

다음 인터페이스를 사용해주세요.

interface Lesson {
id: number;
title: string;
category: string;
level: string;
question: string;

intro: string;

};

interface Problem {
id: string;

}

# 정답 판정

MVP에서는 다음 방식으로 정답을 판정해주세요.

- 숫자형 문제는 Number로 변환해서 비교
- 소수 오차는 0.001까지 허용
- 텍스트형 문제는 공백을 제거하고 비교
- 객관식 문제는 선택한 값과 정답 문자열 비교

수식 자동 동치 판정은 아직 구현하지 말고, 나중에 확장하기 쉽게 judgeAnswer 함수를 분리해주세요.

# 중요한 요구사항

1. 문제만 있는 앱이 아니라 개념 설명이 충분한 앱이어야 합니다.
2. 모든 단원은 오늘의 질문으로 시작해야 합니다.
3. 공식은 단순히 던지지 말고 직관 설명과 함께 보여주세요.
4. 문제를 틀리면 오답 원인을 설명해주세요.
5. 사용자가 전체 학습 흐름을 이해할 수 있도록 다음 단원과의 연결을 반드시 넣어주세요.
6. 모든 UI 문구, 설명, 버튼, 오류 메시지, 콘솔 메시지는 한국어로 작성해주세요.
7. 백엔드는 구현하지 마세요.
8. 모든 데이터는 프론트엔드 내부 TypeScript 객체로 작성해주세요.
9. 코드가 너무 길어지더라도 32단원 전체 데이터를 생략하지 말고 넣어주세요.
10. 그래프 시각화는 이번 버전에서는 필수가 아닙니다. 대신 나중에 추가할 수 있도록 MathText와 FormulaCard 컴포넌트를 분리해주세요.
